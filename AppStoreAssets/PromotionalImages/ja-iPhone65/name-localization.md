# MindVault 日本語名称方針

## 推奨

日本語ローカライズ後も、App Store 上のアプリ名は `MindVault` のまま維持する。

`MindVault` は短く、アプリ内 UI とも一致しており、無理にカタカナ化するよりブランドとして締まって見える。日本語では、アプリ名そのものではなく、サブタイトル、説明文、紹介画像でカテゴリと価値を補足する方が自然。

- アプリ名: `MindVault`
- App Store サブタイトル: `思考をつなぐ知識グラフノート`
- 紹介画像上の短い肩書き: `知識グラフノート`
- 1枚目の見出し: `考えのつながりが見える`

App Store の名前欄はブランド名として簡潔に残し、日本語検索やカテゴリ理解に必要な語はサブタイトルとスクリーンショットに寄せる。

## 検討した代替案

### `MindVault - 知識グラフ`

今回は採用しない。検索語としては分かりやすいが、名称欄が説明的になりすぎ、ブランドの余白が弱くなる。

### `マインドヴォルト`

今回は採用しない。日本語として読める一方、ホーム画面上で長く、英字の `MindVault` よりもプロダクト感が薄くなる。

### `思考の保管庫`

採用しない。意味は伝わるが、少し重く、現代的なノートアプリの軽さから離れる。

## 紹介画像のトーン

`小さなありがとう日記` を参考に、短く安心感のある日本語見出し、1枚1価値、やわらかい余白を使う。ただし MindVault は作業用ノートアプリなので、かわいさよりも「静かで信頼できる」「考えを整理できる」印象を優先する。

1. グラフ起点: `考えのつながりが見える`
2. 書くほど育つ: `メモを書くほど育つ`
3. ローカル検索: `ローカルノートに質問`
4. 安心感: `ローカル優先で安心`
