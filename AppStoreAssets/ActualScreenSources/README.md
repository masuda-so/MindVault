# Actual Screen Sources

This folder contains raw Simulator screenshots used by `Scripts/generate_attached_screen_style_previews.py`.

- `MindVault/iPhone17Pro/`: captured from the MindVault app running on the iPhone 17 Pro simulator.
- `SmallThanksDiary/iPhone17Pro/`: captured from the GratefulMoments / 小さなありがとう日記 app running on the iPhone 17 Pro simulator with `GRATEFUL_MOMENTS_SAMPLE_DATA=1`.

These files are source captures only. The App Store-ready 1284 x 2778 images are generated into `../AttachedScreenStylePreviews/`.
